Créer un formulaire simple avec HTML5 & CSS3

1. Informations Générales

Prerequis

  • Savoir créer un document HTML5
  • Effectuer les contrôles de base avec CSS3
  • Connaître les propriétés basiques de CSS3

Objectifs

  • Maîtriser la conception de formulaires simples
  • Comprendre les règles de base du graphisme appliquées au Web design
  • Savoir comment contrôler les dimensions des éléments HTML avec les propriétés CSS width et height

Concepts Fondamentaux

  • HTML5 | input text
  • HTML5 | input radio
  • HTML5 | input checkbox
  • CSS3 | système de couleurs
  • CSS3 | width
  • CSS3 | height
  • CSS3 | Unités de mesure
  • CSS3 | margin
  • CSS3 | padding
  • CSS3 | border-radius

Recommandations

  • Unités de mesure: pixels(px) pour les dimensions, point(pt) pour la typographie
  • Dimensions: form(250px) sur moblie, et 400px sur Desktop; Image logo(100px) sur Mobile et Desktop
  • Editeurs de texte: Sublime Text 3 ou VS Code
  • Navigateurs: Chrome v84+, Mozilla Firefox v63+, Microsoft Edge v84+
