Créer l' interface d' une calculatrice avec HTML5 & CSS3

1. Informations Générales

Prerequis

  • Savoir créer un document HTML5
  • Effectuer les contrôles de base avec CSS3
  • Connaître les propriétés basiques de CSS3

Objectifs

  • Maîtriser l' usage des balises <div>
  • Comprendre le système des couleurs avec CSS3
  • Comprendre l' utilité des classes
  • Comprendre l' utilité des attributs id
  • Savoir comment contrôler les dimensions des éléments HTML avec les propriétés CSS width et height

Concepts Fondamentaux

  • HTML5 | <button>
  • HTML5 | <div>
  • HTML5 | class
  • HTML5 | disabled
  • HTML5 | Sélecteurs classes multiples
  • HTML5 | id
  • CSS3 | Linear-gradient
  • CSS3 | Système de couleurs
  • CSS3 | width
  • CSS3 | margin
  • CSS3 | padding
  • CSS3 | borders

Recommandations

  • Unités de mesure: pixels(px) pour les dimensions, point(pt) pour la typographie
  • Dimensions: Calculatrice: 300px pour Desktop, 275px pour Smartphone en longueur(width)
  • Editeurs de texte: Sublime Text 3 ou VS Code
  • Navigateurs: Chrome v84+, Mozilla Firefox v63+, Microsoft Edge v84+
