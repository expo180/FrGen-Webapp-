Créer un emploi du temps attrayant avec HTML5 & CSS3

1. Informations Générales

Prerequis

  • Savoir créer un document HTML5
  • Effectuer les contrôles de base avec CSS3
  • Connaître les propriétés basiques de CSS3
  • Savoir ajouter des marges à l' intérieur et à l' extérieur des éléments HTML

Objectifs

  • Savoir concevoir des tableaux avec HTML5
  • Comprendre les règles de base du graphisme appliquées au Web design
  • Savoir styler correctement un tableau avec CSS3
  • Savoir manipuler du texte avec CSS3

Concepts Fondamentaux

  • HTML5 | balise <table>
  • HTML5 | balise <thead>
  • HTML5 | balise <tbody>
  • HTML5 | balise <th>
  • HTML5 | balise <tr>
  • HTML5 | balise <td>
  • CSS3 | Les classes
  • CSS3 | border-radius
  • CSS3 | text-align
  • CSS3 | border-collapse

Recommandations

  • Unités de mesure: pourcentages(px) pour les dimensions, point(pt) pour la typographie
  • Dimensions: 100% pour tous les périphériques
  • Editeurs de texte: Sublime Text 3 ou VS Code
  • Navigateurs: Chrome v84+, Mozilla Firefox v63+, Microsoft Edge v84+
